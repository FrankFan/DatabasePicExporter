## Database Pic Exporter v1.0  数据库结构生成器

##### 功能介绍
* 根据数据库表结构生成图片
* 适用于MS Sql Server 2000, Sql Server 2005 and Sql Server 2008  etc.
* 根据系统sql语句导出数据库表结构
* 利用GDI+将表结构生成图片
